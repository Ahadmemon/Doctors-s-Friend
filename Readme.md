# Doctor’s Friend Web

A web-based Electronic Health Record (EHR) and patient appointment management system developed using the MERN Stack. The platform helps doctors manage patient records, appointments, visit scheduling, and patient communication efficiently.

---

## 📱 Overview

Doctor’s Friend Web is designed to simplify clinic and hospital workflows by providing doctors with a centralized platform for patient data management, appointment handling, and follow-up scheduling.

The system allows patients to book appointments online, while doctors can manage bookings, schedule visits, and send automated SMS reminders to patients.

---

## ✨ Features

### 👨‍⚕️ Doctor Features

- Secure authentication using JWT
- Add, edit, and delete patient records
- Manage patient visit history
- Store treatment details
- Accept or reject appointment requests
- Schedule next visits for patients
- Search patients quickly
- Dashboard for appointment management

### 📅 Appointment System

- Patients can book appointments online
- Doctors can approve or reject bookings
- Appointment status management
- Visit scheduling system

### 📩 SMS Notifications

- Automated SMS reminders using Twilio
- Appointment-related notifications

### 📋 Patient Management

- Store patient personal details
- Maintain medical history
- Track previous treatments
- Organize follow-up visits
